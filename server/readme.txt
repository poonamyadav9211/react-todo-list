
1. create new "server" folder.
2. command: git init
3. make "app.js" as entry file when run "init" command
4. install dependencies : command: "npm install express jsonwebtoken"
5. install server form run application : command: "npm install -g nodemon"
6. add app.js file: command: "touch app.js"

7. run server: command: "nodemon"
8. test api using postman, e.g. "http://localhost:5000/api" and check api is work or note
